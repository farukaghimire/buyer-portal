# Buyer Portal Take-Home Assessment

## Overview
A simple buyer portal for a real estate broker with authentication, favourites management, property search, and detailed property view.

Features:
- User authentication with login/logout.
- View available properties.
- Add/remove properties to/from favourites.
- Search properties by name.
- Click on a property to see detailed information (description, price, location) in a modal popup.
